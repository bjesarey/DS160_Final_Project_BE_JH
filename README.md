# DS160_Final_Project_BE_JH
This project uses the Integrated Postsecondary Education Data System (IPEDS) dataset to analyze characteristics of colleges and universities in the United States. The dataset contains institutional information such as enrollment statistics, admission rates, tuition costs, graduation rates, geographic location, degree offerings, and student demographics. The goal of this project is to build a predictive model using institutional and student-related variables to classify or predict an educational outcome. Potential targets include admission rate, graduation rate category, institutional control type (public/private), or enrollment category. Exploratory data analysis (EDA) was performed to identify missing values, unusual distributions, and relationships among variables. Several machine learning models were tested using different train-test splits and feature sets. The results were evaluated using classification metrics such as accuracy, precision, recall, and confusion matrices. The project demonstrates how educational datasets can be used to identify institutional patterns and improve predictive analysis.
